# 消息字典

## 短信消息队列

- 队列名：sms
- 消息名/消息类型：code | message
- 消息体：`{to: "手机号", msg: ""}`

## 邮件消息队列

- 队列名：mail
- 消息名/消息类型：code | message
- 消息体：`{to: "邮箱地址", msg: ""}`

## DNS 消息队列

- 队列名：dns
- 消息名/消息类型：modify | add | delete
- 消息体：`{record: ["二级域名", "cname | txt", "记录值"]...}`

注：删除时只列出待删除的二级域名列表即可。

## CDN 消息队列

- 队列名：cdn-厂商名
- 厂商名：ali baidu | tengxun | huoshan | baishan | huawei | 
- 消息名/消息类型：modify | add | delete
- 消息体：`{}`
