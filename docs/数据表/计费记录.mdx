# BillingRecord（计费记录）

| 名称        | 类型     | 描述       |
| ----------- | -------- | ---------- |
| user        | -        | 关联的用户 |
| usageAmount | bigint   | 使用量     |
| amount      | decimal  | 总价       |
| billTime    | datetime | 计费时间   |
|             | -        | 单价       |
| status      | enum     | 结清状态   |
|             | string   | 附加描述   |



这里是按量计费的记录
