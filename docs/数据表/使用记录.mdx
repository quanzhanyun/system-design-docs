# ResourceRecord （使用记录）

资源包的使用记录。

| Name        | Type                | Relation With | Description            |
|-------------|---------------------|---------------|------------------------|
| domain      | Relation            | CDN Domain    | 关联的[域名](./域名.mdx)      |
| cdnPackage  | Relation            | CDN Package   | 关联的[资源包](./CDN资源包.mdx) |
| usageAmount | Number(big integer) |               | 使用量以 byte 为单位          |
| usageTime   | Date(datetime)      |               | 使用时间                   |
| area        | Enumeration         |               | 使用区域                   |
