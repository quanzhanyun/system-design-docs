# ResourceRecord （使用记录）

## 字段

| 名称            | 类型     | 描述         |
| --------------- | -------- | ------------ |
| domain          | -        | 关联的域名   |
| resourcePackage | -        | 关联的资源包 |
| usageAmount     | bigint   | 使用量       |
| usageTime       | datetime | 使用时间     |
| area            | enum     | 使用区域     |
