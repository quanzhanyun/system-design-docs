# CDN Bill（CDN 扣费记录）

| Name        | Type        | Relation With | Description |
| ----------- | ----------- | ------------- | ----------- |
| usageAmount | Number      |               | 使用量(单位 byte)         |
| billTimeEnd    | DateTime    |               | 计费结束时间        |
| billTimeStart    | DateTime    |               | 计费起始时间        |
| billStatus  | Enumeration |               | 结清状态,可取值为:  hold(锁定) \| paid(已支付)      |
| user        | Relation    | User          |             |
| domainName  | Text(short text)    |           |计费的域名             |


统计使用量并从 User.cdnBalance 字段中扣流量或 User.cdnFrozenBalance 字段中预扣费。

# 计费方案

参见 [统计与计费](../消息队列/统计与计费.mdx) 。

另外，预扣费时，在 User.cdnFrozenBalance 字段中记录统计时间段内预扣费的总流量，并设置 billStatus 为 hold 状态，当实际扣费时，重新生成一条记录，并且将 billStatus 设置为 paid 状态，预扣费和实际扣费可能不是同一个时间段。

总之，本表中的记录作为回溯流水的记录、凭证。
