# ResourcePackage（资源包）

## 字段

| 名称    | 类型     | 描述        |
| ------- | -------- | ----------- |
| name    | string   | 资源包名称  |
| type    | enum     | 资源包类型  |
| total   | bigint   | 资源包总量  |
| area    | enum     | 加速区域    |
| effTime | datetime | 生效时间    |
| expTime | datetime | 失效时间    |
| remain  | bigint   | 剩余量      |
| user    | -        | 关联的 User |
| status  | enum     | 资源包状态  |


流量进制 1024
