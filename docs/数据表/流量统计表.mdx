# Statistics 流量统计表

流量量统计表。

| Name        | Type                | Relation With | Description         |
|-------------|---------------------|---------------|---------------------|
| domain      | Text(short text)    |               | 关联的域名               |
| date        | Date                |               | 统计日                 |
| fiveMinutes | JSON                |               | 5分钟颗粒度统计            |
| hour        | JSON                |               | 1小时颗粒度统计            |
| day         | Number(big integer) |               | 一天的总使用量（以 byte 为单位） |


流量单位 b (字节) 1000 进制
