# CDN Package（CDN 资源包）

## 字段

| Type      | Relation With  | Description |
| --------- | -------------- | ----------- |
| name      | Text         |             | 资源包名称 |
| type      | Enumeration    |             | 资源包类型 |
| total     | Number(int)    |             | 资源包总量（以 byte 为单位） |
| area      | Enumeration    |             | 加速区域 |
| effTime   | Date(datetime) |             | 生效时间 |
| expTime   | Date(datetime) |             | 失效时间 |
| remain    | Number(int)    |             | 剩余量（以 byte 为单位） |
| user      | Relation       | User        |  |
| pkgStatus | Enumeration    |             | CDN资源包状态 |
