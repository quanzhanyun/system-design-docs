# 域名管理

该功能属于超级管理员及管理员权限。

## 列出所有域名

- 按分页方式列出所有域名
- 根据用户昵称、实名名称、邮箱地址、手机号模糊搜索匹配所有用户并按用户列出所有域名
- 根据字符串搜索所有匹配的域名

## 停用/启用域名

在域名列表中能直接通过开关停用或启用域名。

## 修改域名详细配置

在域名列表中能通过链接直接登录进（无需密码）用户后台的域名配置页面。

## 手动同步域名信息

此场景针对：管理员在上游后台帮用户修改了配置，但此时配置并未同步到我们的用户侧后台，因此需要向消息队列发送一个请求，将上游的配置完整同步到我们的用户侧后台。
