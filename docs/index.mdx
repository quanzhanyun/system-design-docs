:::tip
所有功能模块分为三类：
- 前台（绿色部分）：包括 Nextjs/UI、API
- 中台（蓝色部分）：消息队列、strapi/数据管理
- 后台：各个上游接口、定期执行的任务、数据库
:::

:::info
API 的目的是用于代理系统，等未来规划、实现！
:::
